Redis 基础 ​

背景 ​

mysql 是通过硬盘来存储信息的，并且还要解析并执行 sql 语句，这些决定了它会成为性能瓶颈。而 redis 则是基于内存存储数据的，它可以提供更快的读写速度，并且支持数据的持久化，可以用于缓存，消息队列等场景。
